Exercise 1: Basic Image Insertion

The alt attribute is important for images because it provides alternative text that is displayed when the image cannot be rendered.
This is crucial for accessibility, allowing users who rely on screen readers or have images disabled to understand the content conveyed by the image.

Exercise 2: Image Formats and Attributes

The difference between the real size of the image and the attributes like width and height is that the attributes specify the display size of the image on the webpage, affecting its appearance but not its actual dimensions.
Adjusting these attributes can help optimize the layout and performance of the webpage.
